User acceptance plan

A user acceptance plan divided into 8 sections for evaluating different parts and features of our app

Test ID Test case Description
1 Complete a treasure hunt quiz Start an available treasure hunt, enter your username and manage to complete it until the end, view your score, ranking and leaderboard
2 Continue a treasure hunt While playing a treasure hunt navigate to list of treasure hunts and try to continue your progress
3 Already existing username Try finishing a treasure hunt and entering the same username to start it again
4 QR scanner Try scanning a core code and copying the text displayed to the keyboard
5 Buttons and navigation Evaluate the button design and ease of usage for answering questions and navigation links to move to another page
6 App Design Evaluate the general design of the app
7 End of timer Let the timer reach to zero to see if the treasure hunt ends
8 Download the app on desktop/phone Try downloading the application on your pc or mobile phone
